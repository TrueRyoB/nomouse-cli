# Nomouse CLI
競技プログラマのようなファイルの作成・実行・コピーを迅速に行う必要がある方に向けたCLIサービスです。

## Commands
**Gen**:テンプレートに則した新しいファイルを作成
~~~
nms gen $filename.extension
~~~
**Set**:拡張子に応じたテンプレートを設定
~~~
nms set $extension
~~~
**Run**:コンパイルして実行
~~~
nms run $filename.extension
~~~
**Wind**:最後にnomouse-cliを通じて作成/実行されたファイルのコードをclipboardに保存
~~~
nms wind
~~~
